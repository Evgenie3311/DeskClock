# DeskClock
Сделай свой второй телефон не бесполезным благодаря выводу времени, даты и состояния батареи на экран.

# Компиляция
Скачать и распаковать, использовать Unity 6, выбрать Android.

# Примечание
❗ Проект является простой заглушкой, созданной на досуге для собственного телефона, не рассчитан на оптимизацию, не конкурентоспособен с другими приложениями, ибо движок Unity для игр, а не для каких-то копеечных приложений, которые стоило бы и в Android Studio создать.

❗ Язык месяца зависит от системного языка, если стоит английский, то будет показывать 22 December как в примере ниже.
